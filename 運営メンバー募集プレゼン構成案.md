# 📢 【運営メンバー募集用】プレゼンテーション構成案（大学生向け）

本資料は、学生AIハッカソン（80名規模）のコア運営メンバー（事務局メンバー）を募集するための説明会用プレゼン資料の構成案です。
聞き手である大学生が「ワクワク感」を抱き、当事者として「自分にもできそう！」「0からこの熱狂を一緒に創り上げたい！」と確信できるようなストーリー設計にしています。

今回の改訂では、発起人である田窪氏と三品氏の対談から生まれた**「一人一人が持つ小さなアイデアを、AIの力で社会に実装するプラットフォーム組織の創設」**という壮大なビジョンを背骨として組み込みました。単なる「イベントのスタッフ」ではなく、**「0から新しい組織と文化を創り上げ、創業期メンバーとして全力で走り抜ける」**圧倒的な当事者意識とワクワク感を惹きつける、究極の9スライド構成です。

---

## 🎨 プレゼンの基本コンセプト
- **ターゲット**: AIやビジネス、プロジェクト運営、デザイン、広報などに興味がある大学生（**文系・理系大歓迎！文理不問・学年不問**）
- **トーン＆マナー**: 熱量が高く、カジュアルで話しやすい雰囲気。専門用語は避け、直感的に伝わる言葉を使用する。
- **所要時間想定**: 約15〜20分（質疑応答・座談会を除く）

### 🔵 Slide 5: 運営メンバーは何をするの？（業務イメージ ＆ ぶっちゃけ解説）
- **スライドタイトル**:
  **「難しそうに見えて、実はシンプル！6つの役割の『ぶっちゃけやること』」**
- **スライド内容**:
  - 🛠 **① 統括PM (プロジェクトマネジメント)**
    * ぶっちゃけ：カレンダーを見ながら「みんな進んでる〜？」と優しく声をかけるだけ！
  - 🎤 **② MC / ファシリテーター**
    * ぶっちゃけ：台本（すでにあります！）を元気よく読んで、みんなで拍手するだけ！
  - 💻 **③ テックリード**
    * ぶっちゃけ：プログラミングの基礎知識を活かして、参加チームの技術的な相談役になるだけ！
  - 🎨 **④ デザイン / コンテンツ**
    * ぶっちゃけ：無料のおしゃれツール「Canva」のテンプレートに、文字や写真を当てはめるだけ！
  - 📣 **⑤ コミュニティ / 広報**
    * ぶっちゃけ：Connpassでイベントを作ったり、友達に「こんな面白い勉強会あるよ！」とLINEで共有するだけ！
  - 🍱 **⑥ 会場 / ロジスティクス**
    * ぶっちゃけ：美味しいお弁当屋さんを探して注文し、当日テーブルに並べるだけ！
- **スライドの狙い**:
  専門用語の裏にある「実際の作業レベルのシンプルさ」をカジュアルに提示することで、「これなら自分にもできる！」と安心させ、応募意欲を一気に引き上げる。
- **🗣 トークスクリプト**:
  > 「『PMとか広報とかロジスティクスとか、言葉が難しそうで自分にできるか不安…』と思ったみなさん。
  > **ぶっちゃけ、やることはめちゃくちゃシンプルです！**
  > 例えば『PM』は、スケジュールを見てみんなに『進み具合どう？』と優しく声をかけるだけ。『広報』は、Connpassというイベントページを作ったり、友達に『面白そうなのあるよ！』と送るだけ。『デザイン』は、無料のCanvaというおしゃれなツールで、文字や写真のテンプレートをポチポチ入れ替えるだけです。
  > どうですか？これなら『自分にもできそう！』と思いませんか？難しいことは一切ありません。私たちが普段スマホや日常生活でやっていることの延長線上で、すべての仕事ができるようになっています！」

    * **本ハッカソン**：イベント後もプロダクトが生き続ける！優秀な作品は、私たちのプラットフォーム組織がその後のホスティングや運用をサポートし、世界にデプロイし続ける受け皿を用意します。
  - **② 評価とバトルのリアルさ（Live User Testing）**
    * 普通のハッカソン：綺麗なスライドとピッチの一発芸アイデアだけで審査。
    * **本ハッカソン**：中間発表時にチーム同士が実際にアプリを使い合う「体験テストセッション」を設置。実ユーザーが「本当に使いやすいか」の感情に直撃する、最高のユーザーフィードバックを得られます。

    * 普通のハッカソン：友達同士の内輪ウケか、適当な放置マッチング。
    * **本ハッカソン**：PM（文系・企画）×エンジニア（理系）×デザイナーを運営側が技術・熱量の相性を分析して科学的にマッチング。本物のシリコンバレースタイルの多職種開発チームで0→1を体験できます。
- **スライドの狙い**:
  「大人の介入」や「お金」といったイメージを排除し、ハッカソンそのものの構造的で魅力的な強み（USP）をシャープに示すことで、意識の高い学生を納得させる。
- **🗣 トークスクリプト**:
  > 「大学には山ほどハッカソンやサークルがあります。では、なぜ私たちのプロジェクトなのでしょうか？
