## 🛠️ 技術スタック & 構成

* **フロントエンド**: HTML5, Vanilla JavaScript

* **アセット**: FontAwesome 6.4.0 (アイコン用), Google Fonts (Noto Sans JP, Outfit, Zen Maru Gothic)

### 主要ファイル構造
* `index.html`: LP（ランディングページ）本体。構成および文言の定義。
* `index.css`: レスポンシブ対応、サイバー＆カワイイ独自デザインシステム、アニメーション。
* `index.js`: スクロール連動エフェクト（3Dカードスタック、散乱要素の収束、スクロールインジケータ等）の制御。

## 💻 ローカル開発・検証方法

キャッシュバスターのバージョン管理 (`v=9` 等) を適用しているため、ローカルサーバー経由での閲覧を推奨します。

### 1. ローカルサーバーの起動 (推奨: Python http.server)
プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
python -m http.server 8000
```
起動後、ブラウザで以下のURLにアクセスしてください。
* `http://localhost:8000/index.html?v=9`

### 2. レスポンシブ表示の検証
ChromeやEdgeの開発者ツール（F12）のデバイスシミュレータを用いて、以下の画面サイズで表示を確認してください。
* デスクトップ: `1440 × 900`
* モバイル (iPhone X 等): `375 × 812`

## 🎨 主要なデザイン・レイアウト機能

### 1. 空間突破 (mix-blend-mode ズーム)
* ヒーローセクションへの移行時に、巨大な「DISRUPT」のテキストマスクを突き抜けるような3Dズームイン演出を行います。

### 2. 3Dカードスタック (Timeline)
* ロードマップセクションにおいて、スクロール量に応じてカードが立体的に重なり合い、階層的（階段状）に表示され続ける演出です。

### 3. モバイル横スワイプカルーセル (Challenge Tracks)
* モバイル環境において、課題テーマ（A、B、C）を左右にフリックして切り替える直感的なスワイプUIです。

### 4. 散乱収束演出 (Organizer)
* スクロールに応じて画面外に散らばった16名の運営メンバーカードが、グリッド状に整列・収束していく演出です。
